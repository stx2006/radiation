# radiation 🚀
External radiation source location system simulation.
## 目录结构
radiation<br>
├── computer.py<br>
├── data.py<br>
├── station.py<br>
├── source.py<br>
└── README.md<br>

## 文件说明

### data.py
数据类，用于存储和管理项目中使用的数据。

### source.py
辐射源类，定义了辐射源的属性和行为。

### station.py
测向站类，定义了测向站的属性和行为。

### computer.py
计算机模拟器类，用于模拟测向站的运行和数据处理。

## 安装和使用

### 安装依赖
在项目根目录下运行以下命令以安装所需的依赖项：
```bash
pip install -r requirements.txt
```

### 示例代码
```bash
python test.py
```
